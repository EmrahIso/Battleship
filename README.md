## Run Locally

1. Clone the repository:

   ```bash
   git clone https://github.com/EmrahIso/Battleship.git
   cd Battleship
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the local development server:

   ```bash
   npm run dev
   ```
